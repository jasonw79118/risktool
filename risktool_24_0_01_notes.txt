RiskTool 24.0.01 Notes

Purpose:
- Fix version bouncing/regression caused by old runtime patches still writing 23.0.xx values.
- Fix Portfolio Report duplicate Total Exposure / Residual Risk / Insurance Impact blocks and mismatched totals.

Changed:
1. Version lock
   - index.html now references app_24_0_01.js?v=24.0.01 only.
   - Runtime version display is forced to 24.0.01.
   - Old embedded 23.0.xx / 24.0.xx version strings in the JS runtime were normalized to 24.0.01.

2. Portfolio Report totals
   - Added a final Portfolio Report override module.
   - The top summary cards and scenario ranking table now use the same in-memory row array.
   - Total Exposure = SUM(Total Exposure column from listed scenario rows).
   - Residual Risk = SUM(Residual Risk column from listed scenario rows).
   - Insurance Impact = SUM(Insurance Impact column from listed scenario rows).
   - Removed duplicate visible totals by replacing the Portfolio Report view with one clean report structure.

3. Preserved existing functions
   - Save logic untouched.
   - Scenario workspace architecture untouched.
   - Portfolio Review, View Outcomes, and Export CSV buttons retained.
   - Post-login load workspace flow remains intentionally ignored per current direction.
