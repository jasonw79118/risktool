PowerShell update steps

1. Extract this package into:
   C:\Users\jason\risktool

2. In PowerShell:
   cd C:\Users\jason\risktool

3. Stage everything:
   git add .

4. Commit:
   git commit -m "Add Risk Manager phase 3 scenario builder and Monte Carlo engine"

5. Push:
   git push origin main

6. If the remote is not set yet:
   git init
   git remote add origin https://github.com/YOURUSERNAME/risktool.git
   git branch -M main
   git push -u origin main
