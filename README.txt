Risk Manager – Phase 4.3.1 Surgical Update

This package is a surgical upgrade based on the real 4.2 layout.

Included changes:
- Dashboard open-scenario table for scenarios not closed
- Inherent risk score is calculated, not manually entered
- Auto-generated scenario IDs in YYYYMMDD-00001 format
- Category Admin shows existing selections and allows add/edit/remove
- Category-driven selections are alphabetized
- Test Scenario buttons added to Single and Complex builders
- Information menu with user manual
- Optional custom Monte Carlo JSON configuration loader

Important:
This version still stores scenarios in browser localStorage.

- Monte Carlo report documentation now includes Method, Input, and Output tables
- Custom Monte Carlo JSON can change randomization method and event logic
