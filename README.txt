Risk Manager – Phase 5.3 Navigation + Random Scenarios Patch

This patch adds:
- clickable Scenario ID links in the dashboard open-scenario table
- a Builder column showing whether the scenario came from Single or Complex Scenario
- clicking the ID reopens the scenario in the original builder
- Random Scenarios fields in both Single and Complex Scenario builders
- allowed Random Scenarios values: 100, 500, 1,000, 10,000, 100,000
- issue-name links in the Complex Scenario Risk Items table that highlight the issue row
- Information note for the future beta-distribution scenario menu

This is a focused patch on top of the last confirmed working version.
