RISKTOOL 24.0.03 NOTES

CHANGES MADE
1. Added Test Mode toggle in User Admin.
   - OFF hides Model Validation.
   - ON shows Model Validation in the left pane.

2. Added Model Validation screen shell.
   Buttons added:
   - Load Core Banking Test Portfolio
   - Clear Test Portfolio
   - Download Model Validation Pack
   - Export Validation Dataset

3. Added deterministic Core Banking Test Portfolio loader.
   - Uses TEST-CORE-### IDs.
   - Adds 8 deterministic bank/core-style validation scenarios.
   - Includes risk items, costs/losses, evidence, mitigations, insurance, accepted risk, and Monte Carlo rows.

4. Standardized Monte Carlo run options.
   - Allowed: 100, 1,000, 10,000, 100,000.
   - Removed 500 option from runtime and HTML selects.

5. Version moved to 24.0.03 and locked in the runtime.

NOT CHANGED
- Portfolio Report calculation module was not redesigned in this patch.
- Save logic was not changed.
- Post-login workspace load screen was not changed.
- Workspace architecture was not changed.

REGRESSION TESTING
